#                           Wireshark Network Traffic Analysis & Credential Capture
# Overview
This project demonstrates network traffic analysis using Wireshark, including HTTP inspection, TCP handshake analysis, and capturing login credentials from an insecure web application.
# Key Concepts
Packet Sniffing
HTTP Traffic Analysis
TCP 3-Way Handshake
Credential Capture
# Tools Used
Wireshark
Web Browser

# Key Finding
Captured plaintext credentials from HTTP POST request showing insecurity of unencrypted communication.

This project proves why HTTPS is essential for securing user data.
